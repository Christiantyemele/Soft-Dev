# AgentFlow TUI Installer & Guided Setup Wizard — Implementation Plan

## Overview

Add a clean TUI (Terminal User Interface) installer and setup experience to AgentFlow, modeled after OpenClaw's `install.sh` + `openclaw onboard` + `openclaw tui` flow. Users will be able to install, configure, and monitor AgentFlow entirely from the terminal with a guided, interactive experience.

```rust
pub fn init_tui() -> Result<Terminal<CrosstermBackend<std::io::Stdout>>> {
    crossterm::execute!(std::io::stdout(), EnterAlternateScreen, EnableMouseCapture)?;
    crossterm::terminal::enable_raw_mode()?;
    let backend = CrosstermBackend::new(std::io::stdout());
    Terminal::new(backend)
}

pub fn restore_tui() {
    crossterm::terminal::disable_raw_mode().ok();
    crossterm::execute!(std::io::stdout(), LeaveAlternateScreen, DisableMouseCapture).ok();
}
```

#### 2.7 `.env` File Generation

```rust
pub fn write_env_file(config: &SetupConfig, project_dir: &Path) -> Result<()> {
    let mut content = String::new();
    content.push_str("# Generated by agentflow setup\n");
    content.push_str(&format!("ANTHROPIC_API_KEY={}\n", config.anthropic_key));
    content.push_str(&format!("GITHUB_PERSONAL_ACCESS_TOKEN={}\n", config.github_pat));
    content.push_str(&format!("GITHUB_REPOSITORY={}\n", config.repo));
    if let Some(ref key) = config.gemini_key {
        content.push_str(&format!("GEMINI_API_KEY={}\n", key));
    }
    if let Some(ref key) = config.openai_key {
        content.push_str(&format!("OPENAI_API_KEY={}\n", key));
    }
    if config.proxy_enabled {
        content.push_str(&format!("PROXY_URL={}\n", config.proxy_url));
        content.push_str(&format!("PROXY_API_KEY={}\n", config.proxy_api_key));
        content.push_str(&format!("GATEWAY_URL={}\n", config.gateway_url));
        content.push_str(&format!("GATEWAY_API_KEY={}\n", config.gateway_api_key));
    }
    content.push_str(&format!("AGENTFLOW_WORKSPACE_ROOT={}\n", config.workspace_dir));
    content.push_str("RUST_LOG=info,agent_team=debug,pocketflow_core=debug\n");

    std::fs::write(project_dir.join(".env"), content)?;
    Ok(())
}
```

### Phase 5: Install Script (`scripts/install.sh`)

#### 5.1 Script Features

```bash
#!/usr/bin/env bash
# AgentFlow Installer
# Usage: curl -fsSL https://get.agentflow.dev | bash
#   or:  curl -fsSL https://raw.githubusercontent.com/The-AgenticFlow/AgentFlow/main/scripts/install.sh | bash

set -euo pipefail

# Detect OS and architecture
detect_platform() {
    local os arch
    os=$(uname -s | tr '[:upper:]' '[:lower:]')
    arch=$(uname -m)
    case "$arch" in
        x86_64) arch="x86_64" ;;
        aarch64|arm64) arch="aarch64" ;;
        *) echo "Unsupported architecture: $arch"; exit 1 ;;
    esac
    echo "${os}-${arch}"
}

# Check/install Rust toolchain
ensure_rust() {
    if ! command -v rustc &>/dev/null; then
        echo "Rust not found. Installing rustup..."
        curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y
        source "$HOME/.cargo/env"
    fi
    echo "✓ Rust $(rustc --version)"
}

# Check/install Git
ensure_git() {
    if ! command -v git &>/dev/null; then
        echo "✗ Git is required. Please install git first."
        exit 1
    fi
    echo "✓ Git $(git --version)"
}

# Check/install Node.js
ensure_node() {
    if ! command -v node &>/dev/null; then
        echo "Node.js not found. Installing via nvm..."
        curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
        source "$HOME/.nvm/nvm.sh"
        nvm install 20
    fi
    echo "✓ Node.js $(node --version)"
}

# Build from source (fallback) or download binary
install_agentflow() {
    local platform=$(detect_platform)
    local install_dir="${AGENTFLOW_INSTALL_DIR:-$HOME/.local/bin}"
    mkdir -p "$install_dir"

    # Try to download pre-built binary from GitHub Releases
    if command -v gh &>/dev/null; then
        local latest=$(gh release view --repo The-AgenticFlow/AgentFlow --json tagName -q .tagName)
        echo "Downloading AgentFlow ${latest} for ${platform}..."
        gh release download "$latest" --repo The-AgenticFlow/AgentFlow --pattern "*${platform}*" --output "$install_dir/agentflow"
        chmod +x "$install_dir/agentflow"
        echo "✓ Installed to $install_dir/agentflow"
    else
        # Fallback: build from source
        echo "Building AgentFlow from source..."
        local repo_dir=$(mktemp -d)
        git clone https://github.com/The-AgenticFlow/AgentFlow.git "$repo_dir"
        cd "$repo_dir"
        cargo build --release
        cp target/release/agentflow "$install_dir/agentflow"
        cp target/release/agentflow-setup "$install_dir/agentflow-setup"
        cp target/release/agentflow-dashboard "$install_dir/agentflow-dashboard"
        cp target/release/agentflow-doctor "$install_dir/agentflow-doctor"
        chmod +x "$install_dir"/agentflow*
        rm -rf "$repo_dir"
        echo "✓ Built and installed to $install_dir/"
    fi
}

# Offer to run setup wizard
run_setup() {
    echo ""
    echo "Would you like to run the setup wizard now? (Y/n)"
    read -r response
    if [[ "$response" =~ ^[Yy]$ ]] || [[ -z "$response" ]]; then
        agentflow-setup
    fi
}

# Main
main() {
    echo "╔══════════════════════════════════════════════╗"
    echo "║       AgentFlow Installer                    ║"
    echo "╚══════════════════════════════════════════════╝"
    echo ""

    ensure_rust
    ensure_git
    ensure_node
    install_agentflow
    run_setup

    echo ""
    echo "Installation complete!"
    echo "Run 'agentflow setup' to configure, or 'agentflow tui' to monitor."
}

main "$@"
```

## Risks & Mitigations

| Risk | Mitigation |
|------|-----------|
| `tui-input` crate API changes | Pin exact version, add integration test |
| Crossterm raw mode conflicts with existing tracing | Use `tracing-subscriber` with `fmt` layer that respects alternate screen |
| Install script fails on edge OS/arch | Graceful fallback to source build with clear error message |
| Dashboard polling too frequent | Configurable refresh interval, default 2s |
| `.env` file overwrites existing config | Prompt user with `[Overwrite] [Merge] [Cancel]` |
